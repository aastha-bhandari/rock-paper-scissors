# rock-paper-scissors game
a simple beginner project of rock paper scissors game made using python library Tkinter

# Technologies Used -
Python 3 <br>
Tkinter (GUI Library) <br>
Random Module

# Games Rules -
Rock beats Scissors <br>
Scissors beats Paper <br>
Paper beats Rock <br>
If both player and the computer choose the same option, the round ends in draw.
